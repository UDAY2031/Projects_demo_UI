Deep Dives into VisuWeave and EcoVista

VisuWeave

Python, ASR, NLP (BERT), GANs, RL

Dynamic Presentation System

A real-time Speech-to-Visual mapping system. It utilizes cloud-based ASR for transcription and NLP models (like BERT) to extract critical phrases and context from live audio. This is followed by a pipeline leveraging AI image generation and web scraping to dynamically produce visuals and animated transitions in a GUI, ensuring visual coherence and minimal latency.

EcoVista

Python, OpenCV, YOLOv8, IoT, CVAT

AI-Driven Waste Segregation

A comprehensive waste management solution with four modules: YOLO v8-based Segregation (dry, wet, recyclable, hazardous), CVAT-enhanced Litter Detection, a cost-effective Hardware Prototype using OpenCV for classification, and a User/Admin Application. The system includes an app-based reward system and surveillance monitoring to encourage proper disposal.
