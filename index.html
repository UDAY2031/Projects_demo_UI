EcoVista - AI-Driven Waste Segregation

Tech Stack: Python, OpenCV, YOLOv8, IoT-enabled Hardware, CVAT

A comprehensive waste management solution with four modules: **YOLO v8-based Segregation** (dry, wet, recyclable, hazardous), **CVAT-enhanced Litter Detection**, a cost-effective **Hardware Prototype** using OpenCV for classification and sorting, and a **User/Admin Application**. The system includes an app-based reward system and surveillance monitoring with image matching to encourage proper disposal and maintain clean surroundings.
